The purpose of this assignment is to evaluate the accessibility of any website, state the failures by comparing them to WCAG criterion. Afterwards we need to provide solutions for those failures, create a webpage of our own, present our findings and solutions through that webpage in such a way that it is easily accessible for individuals with various disabilities. The website I selected for this assignment/project was Lingscars . This was a website having various accessibility issues.

Error 1

Design based: Background which can interfere witht the actual website content.

Fails WCAG Success Criterion 1.4.11 Non-text Contrast

Confusing and too colorful backgrounds can cause multiple issues. For a contrast ratio of an image to be consistent, the background image or color must not be presented in such a chaotic manner. It may interfere with the website content, and can lead to difficulties for individuals with visual disabilities.


---------------


Error 1 solution

Give the background either a solid color or a mind texture, so that it does not interfere with the actual content of the page in any way.

>
Error 2

Content based: No alternate text

Fails WCAG Success Criterion 1.1.1 Non-text Content

If an image doesn't load properly, there is no way for a visually challenged person to know the context.


---------------


Error 2 solution

Add a basic description of every image in the alt tag of that image.

<img src="img1.gif" alt="Man holding rental sign" height="42" width="42">

<img src="img2.gif" alt="Toyota Aygo car image" height="42" width="42">

Error 3

Code based: No structural elements

Fails WCAG Success Criterion 2.4.10 Section Headings

Makes it harder for various ATs to properly convey information to the user.


---------------


Error 3 solution

Add appropriate heading tags wherever there is a new section, to give the content a hierarchy.

<h1>LINGS CARS>

<p>Description of the website of Lingscars.>

<h2>Toyota Aygo>

<p>Hatchback 5dr 998cc Petrol Manual>

<h3>Fuel>

<p>Petrol>

<h3>Paint>

<p>Non-metallic>

<h3>Gearbox>

<p>Manual>
